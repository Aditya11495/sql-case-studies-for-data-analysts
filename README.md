# SQL Data Analyst Interview Questions

This repository contains commonly asked **SQL interview questions for Data Analyst roles** along with example datasets and solutions.

The goal of this repository is to practice SQL concepts frequently used in real-world analytics problems and technical interviews.

## Topics Covered

* SQL Basics
* Filtering Data
* Aggregations (GROUP BY, HAVING)
* Joins
* Subqueries
* Window Functions
* Ranking Functions
* Analytical Queries

## Example Problems

Some of the SQL problems included:

1. Find duplicate records
2. Second highest salary
3. Find NULL values
4. Employee department join
5. Customers without orders
6. Total sales per product
7. Departments with more than 5 employees
8. Customers with multiple orders
9. Latest order per customer
10. Rank employees by salary
11. Running total of sales
12. Monthly revenue
13. Employees earning more than their manager
14. Top 3 salaries per department
